## 启动问题
- 默认 `JAVA_HOME` 指向 Java 8，需指定 `JAVA_HOME=$(/usr/libexec/java_home -v 17)` 启动
- `mvn spring-boot:run` 不加参数时 Tomcat 端口变成 63122（非 8080），需加 `-Dspring-boot.run.arguments="--server.port=8080"` 显式指定
- `start.sh` 中启动后端需要在 backend 目录下执行，否则找不到 spring-boot 插件
